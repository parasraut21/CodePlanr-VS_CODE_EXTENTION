🤖 CODEPLANR AI - SUPER SIMPLE GUIDE

═══════════════════════════════════════════════════════════════

STEP 1: RUN IT
═══════════════════════════════════════════════════════════════
Press F5

═══════════════════════════════════════════════════════════════

STEP 2: OPEN SIDEBAR
═══════════════════════════════════════════════════════════════
Click the 🤖 ROBOT ICON on the left side

═══════════════════════════════════════════════════════════════

STEP 3: SET API KEY
═══════════════════════════════════════════════════════════════
Click ⚙️ CONFIG button at the top
Enter your OpenAI API key
(Get from: https://platform.openai.com/api-keys)

═══════════════════════════════════════════════════════════════

STEP 4: CHAT!
═══════════════════════════════════════════════════════════════
Type anything and press Enter

═══════════════════════════════════════════════════════════════

EXAMPLES
═══════════════════════════════════════════════════════════════

Ask Questions:
- "How do I create a REST API?"
- "Explain promises"
- "Help me debug this"

Create Files:
- "Create a C++ file for addition of 2 numbers"
- "Make a Python script to read CSV"
- "Create a JavaScript email validator"

═══════════════════════════════════════════════════════════════

BUTTONS
═══════════════════════════════════════════════════════════════
⚙️ CONFIG - Set your API key
🗑️ CLEAR - Clear the chat

═══════════════════════════════════════════════════════════════

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Can't see sidebar?
→ Click 🤖 robot icon on left

Buttons not working?
→ Make sure extension is running (Press F5)

"API key not configured"?
→ Click ⚙️ CONFIG button

File creation fails?
→ Open a folder first (File → Open Folder)
